What About Hedgehogs?

"What About Hedgehogs?" was de masterthesis van Ewout De Vos voor zijn opleiding aan LUCA School of Arts. Ik moest gelijktijdig een bachelorproef maken aan de PXL-Music en dit was de perfecte kans om samen te werken.

Zo heb ik voor mijn bachelorproef de game van audio voorzien en zelf de audiosystemen geprogrammeerd.
Ook legde mijn bachelorproef de nadruk op generatieve muziek, muziek die deels door de computer wordt gegenereerd op basis van de gameplay. Zo zijn er een aantal plaatsen waar de muziek veranderd op basis van de afstand tot een object (zoals een kwaadaardige hond).

De muziek in het algemeen verandert van zone tot zone door middel van collision boxes. Tussen alle zones zijn collisions geplaatst die in de muziek de juiste zone triggert. Dankzij de audio effects heb ik veel bijgeleerd over omgaan met iemand anders codebase. Ik heb geprobeerd om de audio code apart te houden om niet in de weg te lopen van Ewout, maar soms is dit uiteraard onvermijdelijk. De code voor de voetstappen kon bijvoorbeeld volledig apart gehouden worden, het enige wat deze code doet is namelijk een raycast uitsturen naar de grond en op basis daarvan speelt het juiste voetstapgeluid. Voor de eetgeluidjes bijvoorbeeld moest ik wel in de code van Ewout duiken.
